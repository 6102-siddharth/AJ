Problem statements

The hotel booking data is messy and difficult to use. 
It has duplicates, missing values, and many unnecessary columns.
Because of this, it is hard to understand bookings, cancellations,and customer behavior.
This project aims to clean the data,remove useless columns,
and prepare a clean dataset so that useful insights and dashboards 
can be created for better business decisions.
